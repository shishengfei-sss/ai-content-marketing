## 0. 执行规则（必须遵守）

### 0.1 一步一验

1. **只允许顺序执行** Step 清单；不得跳过。
2. 每个 Step 完成后必须：
   - 跑该 Step 的 **专项验收**（下表「验收」）→ 全部 PASS
   - 跑 **回归门禁**（§0.3）→ PASS
3. 任一 FAIL → **停止**，修复后从该 Step 重跑，不得进入下一步。

| 标记 | 含义 |

| ⬜ | 未开始 |
| 🔄 | 进行中 |
| ✅ | 本 Step 专项 + 回归均 PASS |
