# v0.5 CRM 缺口补齐 — 逐步验收计划（CRM-2e）

| 项目 | 说明 |
|------|------|
| 文档版本 | v1.0-crm-gap |

| 专项验收 | `python apps/api/tests/verify_crm_2e.py` |
| 单步验收 | `python apps/api/tests/verify_crm_2e.py --step 2e-N` |
| 回归门禁 | 每 Step 后 `VERIFY_SKIP_AGENT=1 python tests/verify_crm_regression.py` |

## 0. 执行规则

1. **顺序执行** 2e-1 → 2e-8，不得跳步。
2. 每 Step 完成後：
   ```bash
   cd apps/api
   python tests/verify_crm_2e.py --step 2e-N
   VERIFY_SKIP_AGENT=1 python tests/verify_crm_regression.py
   ```
3. 全部 Step PASS 后：
   ```bash
   python tests/verify_crm_2e.py
   python tests/run_crm_all.py --through 2e
   ```
4. 更新 `v0.5-crm执行计划.md` §0.6：`2e` → ✅；`verify_crm_final.py` 的 `--through` 改为 `2e`。
