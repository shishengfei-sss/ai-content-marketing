## 1. 总原则

| 原则 | 说明 |
|------|------|
| 先后端、后双端 | API 验收通过后再改 Web/H5 |
| 一步一验 | 当前里程碑全部 PASS 才进入下一步 |
| 服务端为准 | 额度、权限、发布确认在 API 强制 |
| 不破坏 v0.3.3 | 每步跑 `run_m0_m8.py` |
| BR-01 | Agent **禁止**未经用户 confirm 自动 publish |
| FR-REVIEW-10 | 不恢复强制审核流；Compliance 仅建议/拦截 |
